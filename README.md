# SDSU_Planner
CS250 Group Project - Handed off to Group 2.

!! Currently under development !!

How to run:
1. Install Python, which can be found [here](https://www.python.org/downloads/release/python-3140/)
2. Clone the project
3. Install Flask with `pip install flask` in a terminal
4. Open terminal in `.\backend` and run `py .\main.py`
5. Open a web browser and go to https://localhost:5000/
